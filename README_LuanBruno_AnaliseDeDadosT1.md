## Como rodar o projeto completo

### 1. Clone o repositório

```bash
git clone https://github.com/luanbconceicao/mini-projeto-luanbruno-analisedadost1.git
cd mini-projeto-luanbruno-analisedadost1
```

### 2. Crie e ative o ambiente virtual

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Instale as dependências

```bash
pip install -r requirements.txt
```

### 4. Execute o projeto

```bash
python main.py
```
